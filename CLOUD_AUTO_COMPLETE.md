## ⏰ 定时说明

### 当前设置
- **运行时间**：每天北京时间早上8:00
- **时区**：UTC 0:00
- **频率**：每天一次

### 修改时间
编辑 `.github/workflows/daily-auto.yml` 文件：

```yaml
schedule:
  # 当前：北京时间早上8点
  - cron: '0 0 * * *'

  # 改为北京时间早上9点
  # - cron: '0 1 * * *'

  # 改为每天运行两次（早8点和晚8点）
  # - cron: '0 0,12 * * *'
```

保存后提交，GitHub会自动使用新时间。
