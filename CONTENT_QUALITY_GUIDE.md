### 方案2：后处理过滤（辅助）

在生成后自动过滤延伸阅读：

```python
def filter_reading_links(links):
    """过滤延伸阅读链接"""

    # 关键词：应该保留
    keep_keywords = ['司法解释', '典型案例', '指导性案例', '会议纪要', '规定']

    # 关键词：应该删除
    remove_keywords = ['召开', '部署', '讲话', '慰问', '调研']

    filtered = []
    for link in links:
        title = link['title']

        # 检查是否包含保留关键词
        if any(kw in title for kw in keep_keywords):
            filtered.append(link)
            continue

        # 检查是否包含删除关键词
        if not any(kw in title for kw in remove_keywords):
            filtered.append(link)

    return filtered
```

### 方案3：质量评分机制（进阶）

```python
def rate_content_quality(section):
    """对内容质量进行评分"""

    score = 0

    # 深度评分
    if '实务要点' in section or '实务建议' in section:
        score += 3
    if len(section) > 200:  # 内容够长
        score += 2
    if '具体' in section or '案例' in section:
        score += 2

    # 价值评分
    if '策略' in section or '建议' in section:
        score += 3

    return score

# 要求趋势洞察评分 >= 8分
```

## Prompt模板改进建议

### 当前问题分析

GLM-4.7可能生成"安全但空洞"的内容，因为：
1. Prompt没有明确要求深度
2. 缺少具体的写作标准
3. 没有提供好的示例

### 改进的Prompt结构

```python
prompt = f"""
# 角色
你是一位资深的法律实务专家，拥有20年律师执业经验。

# 任务
分析以下法律资讯，生成高质量的每日法律简报。

# 资讯内容
{news_items}

# 输出要求

## 1. 趋势洞察（最重要）

### 深度标准
每个板块必须包含：
- **现象描述**（2-3句话）：发生了什么
- **影响分析**（2-3句话）：对实务有什么影响
- **实务要点**（3-5条）：具体如何应对

### 示例（好的趋势洞察）

**立法动态：新《公司法》注册资本制度变革**

2024年7月1日新《公司法》实施，注册资本认缴制调整为5年内实缴。这一变革对公司设立、股权转让、减资程序产生深远影响。

**实务要点：**
- 新设公司：合理规划注册资本额度和实缴期限
- 存量公司：需要在2027年7月1日前完成实缴或减资
- 股权转让：受让方需注意未实缴出资的连带责任
- 减资程序：严格履行通知债权人和公告义务
- 法律风险：未按期实缴可能面临股东资格丧失

### 禁止出现
❌ "体现了...精神"
❌ "强调了...重要性"
❌ "指出...需要加强"
❌ 其他空洞的表述

## 2. 延伸阅读筛选

### 必须保留（有法律价值）
- 司法解释全文
- 典型案例/指导性案例
- 具体的法律规定
- 有法律适用价值的会议纪要

### 必须删除（无法律价值）
- 一般性工作会议报道
- 领导讲话/调研
- 纯粹的活动新闻
- 行政事务性通知

## 3. 写作风格

- 专业但不过于学术化
- 具体但不罗列细节
- 有案例支撑但不讲故事
- 可操作但不给具体建议

# 输出格式
严格按照以下模板输出：

{template}

# 评分标准（自检）
- 趋势洞察深度：>= 8/10
- 实务价值：>= 8/10
- 延伸阅读价值：>= 8/10

如果某部分内容无法达到标准，请重写。
"""
```
